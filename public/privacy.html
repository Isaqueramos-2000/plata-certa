Política de Privacidade

Última atualização: 12 de maio de 2026

O PlantaCerta foi criado para ajudar você a identificar e cuidar das suas plantas. Sua privacidade é levada a sério e esta política explica, em linguagem simples, quais dados são coletados e como são usados.

1. Dados coletados

O PlantaCerta coleta apenas o mínimo necessário para funcionar:

  • Fotos enviadas para identificação: imagens que você escolhe enviar são transmitidas ao nosso servidor para processamento pela IA. As imagens não são armazenadas no servidor após o processamento.
  • Identificador anônimo de dispositivo: um ID gerado automaticamente, sem nenhuma informação pessoal, usado para controlar o uso da IA e evitar abusos.
  • Dados do jardim (locais): nomes de plantas, fotos, histórico de cuidados e lembretes são salvos exclusivamente no seu dispositivo. Não fazemos upload dessas informações para nenhum servidor.

2. Dados que NÃO coletamos

  • Nome, e-mail ou qualquer dado de identificação pessoal
  • Localização geográfica
  • Contatos, câmera permanente ou microfone
  • Dados de pagamento (o app é gratuito)

3. Como os dados são usados

  • As fotos enviadas são processadas pela IA (Claude, da Anthropic) somente para identificar a planta e gerar o guia de cuidados.
  • O identificador anônimo é usado para limitar o número de identificações por hora e garantir disponibilidade do serviço para todos.
  • Nenhuma informação é vendida, compartilhada ou usada para publicidade.

4. Serviços de terceiros

O PlantaCerta utiliza os seguintes serviços externos:

  • Anthropic (Claude AI): processamento de imagens para identificação de plantas. Política de privacidade da Anthropic.
  • Firebase (Google): autenticação anônima para geração de identificadores únicos por dispositivo. Política de privacidade do Firebase.
  • Vercel: hospedagem do servidor proxy. Política de privacidade da Vercel.

5. Armazenamento e segurança

Todos os dados do jardim (plantas, fotos, histórico) ficam armazenados localmente no seu dispositivo. As fotos são salvas na pasta de documentos do app, inacessível por outros aplicativos. A comunicação entre o app e o servidor usa HTTPS com criptografia.

6. Permissões do aplicativo

  • Câmera: usada somente quando você opta por tirar uma foto para identificação.
  • Armazenamento: para salvar fotos das plantas no seu dispositivo.
  • Notificações: para lembretes de rega que você configura voluntariamente.

7. Crianças

O PlantaCerta não coleta dados de crianças menores de 13 anos. O app não exige cadastro e não armazena informações pessoais identificáveis de nenhum usuário.

8. Seus direitos

Como todos os dados do jardim ficam no seu próprio dispositivo, você tem controle total sobre eles. Pode excluir plantas, histórico e fotos diretamente no app, ou desinstalar o app para remover todos os dados locais.

9. Alterações nesta política

Caso esta política seja atualizada, a nova versão será publicada nesta página com a data de atualização. O uso contínuo do app após alterações implica aceitação da nova política.
